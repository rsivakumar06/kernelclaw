// architecture
Six-step autonomous loop
01
👁
trigger

Watch

A file watcher monitors your GPU code directory. The moment a .py or .cu file is saved, the agent wakes up — no commands, no buttons.

02
📊
tool use

Profile

NVIDIA's ncu profiler runs automatically, measuring SM occupancy, tensor core utilization, memory coalescing, and warp stall reasons at the hardware level.

03
🧠
reasoning

Diagnose

Nemotron reads the dense profiler output and produces a plain-English diagnosis: exactly what bottleneck exists and why.

04
✍️
code gen

Rewrite

Based on the diagnosis, the agent generates a fixed version using CUDA — directly targeting the identified bottleneck.

05
⚡
verification

Benchmark

Both versions run side-by-side, five times each, measuring real wall-clock throughput. If the new version isn't faster, it gets discarded.

06
💾
memory

Remember

Every optimization pattern is written to persistent memory. The agent gets measurably smarter over time.
